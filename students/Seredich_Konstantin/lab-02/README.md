# Проект "Говорю красиво" - Гексагональная архитектура

Архитектурный каркас для сервиса интервального повторения карточек (`StudyService`).
Код разделен на слои согласно паттерну Ports and Adapters:
- `domain`: Бизнес-логика расчета интервалов повторения.
- `application`: Сценарии использования (UseCase) и интерфейсы портов.
- `infrastructure`: Адаптеры (InMemory БД) и DI-контейнер для внедрения зависимостей.
